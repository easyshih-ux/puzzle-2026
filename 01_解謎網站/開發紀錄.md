# 01 解謎網站開發紀錄

更新日期：2026-06-21

## 專案內容

- 開頭頁：`開頭製作/index.html`
- 首頁大廳：`03首頁/home.html`
- 其他首頁備份頁：`03首頁/index_single_fixed.html`

## 已完成項目

### 1. 開頭頁建立

- 新增 `開頭製作` 資料夾。
- 建立 `index.html` 作為開場頁。
- 開場流程為：播放 `01言辵米角刀牛開場動畫v2.mp4`，播放結束後切換到 `課程封面.png`。
- 封面頁可點擊進入 `03首頁/home.html`。

### 2. 課程封面與進場提示

- 封面頁改為完整等比顯示，避免底部內容被裁切。
- 加入「開場動畫結束後，點擊畫面進入首頁」提示文字。
- 加入「若開場動畫沒有自動播放，請先點一下畫面」提示文字。

### 3. 開場影片顯示修正

- 影片由原本滿版裁切，改為置中完整顯示。
- 影片外圍加上框線與留白，視覺呈現比照課程封面模式。
- 目標是保留影片下方字幕，不再因為 `cover` 裁切而消失。

### 4. 影片互動功能

- 影片開始播放後，自動隱藏底部提示文字。
- 新增右上角音樂切換按鈕：
  - `關閉音樂`
  - `開啟音樂`

### 5. 首頁大廳修正

- 修正 `03首頁/home.html` 在平板顯示時主標題 `言辵米．角刀牛` 可能換成兩行的問題。
- 透過放寬中央標題卡、縮小平板字級與調整字距，使主標更穩定維持單行。
- 將首頁下方「任務提醒」區塊往上調整，降低學生忽略的機率。

### 6. 其他補充

- 新增 `course_link_qr.png` 並已納入版本控制。
- 整理可提供封鎖或白名單使用的 GitHub Pages 網址。
